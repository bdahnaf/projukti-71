UVA 11799 Hint:

  • You have to find the biggest value.
  • The first number of each test case is the number of the students acting as scary creatures. So use a for loop from 1 to N.
  • Initialize test=0. Take each input in the for loop & if the input is bigger than test store it’s value to test.
  • Say test=0. Then you start the loop & the 1st input is 5. Then test=5. and the second input is 3. The value of test won’t change this time. Thus you will get the biggest value.
